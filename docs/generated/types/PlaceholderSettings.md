# PlaceholderSettings

Settings for the `HxPlaceholder` and derived components.

## Properties

| Name | Type | Description |
|------|------|-------------|
| Color | `ThemeColor?` | Explicit color of the placeholder. |
| CssClass | `string` | Additional CSS class for `HxPlaceholder`. |
| Size | `PlaceholderSize?` | Size of the placeholder. |
