# AvatarSettings

Settings for the `HxAvatar` and derived components.

## Properties

| Name | Type | Description |
|------|------|-------------|
| Color | `ThemeColor?` | Theme color of the avatar (background and contrasting content color). |
| CssClass | `string` | Any additional CSS class to apply. |
| Size | `AvatarSize?` | Size of the avatar. |
| Subtle | `bool?` | When `true`, the avatar uses the subtle variant of the theme color. |
