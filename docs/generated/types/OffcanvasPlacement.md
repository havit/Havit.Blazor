# OffcanvasPlacement

Represents the placement of an offcanvas.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| End | 0 | Offcanvas is placed at the end. |
| Start | 1 | Offcanvas is placed at the start. |
| Top | 2 | Offcanvas is placed at the top. |
| Bottom | 3 | Offcanvas is placed at the bottom. |
