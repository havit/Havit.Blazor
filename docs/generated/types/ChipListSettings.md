# ChipListSettings

Settings for the `HxChipList` component.

## Properties

| Name | Type | Description |
|------|------|-------------|
| ChipBadgeSettings | `BadgeSettings` | Settings for the `HxBadge` previously used to render chips. |
| Color | `ThemeColor?` | The theme color of the chips (applied as a Bootstrap `theme-*` class). |
| CssClass | `string` | Additional CSS class. |
| ShowResetButton | `bool?` | Enables or disables the reset button. |
