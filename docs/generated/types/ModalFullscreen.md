# ModalFullscreen

Fullscreen behavior for `HxModal`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Disabled | 0 | Fullscreen mode disabled. |
| Always | 1 | Always fullscreen modal. |
| SmallDown | 2 | Fullscreen for viewports below the "small" breakpoint (`576px`). |
| MediumDown | 3 | Fullscreen for viewports below the "medium" breakpoint (`768px`). |
| LargeDown | 4 | Fullscreen for viewports below the "large" breakpoint (`992px`). |
| ExtraLargeDown | 5 | Fullscreen for viewports below the "extra-large" breakpoint (`1200px`). |
| XxlDown | 6 | Fullscreen for viewports below the "XXL" breakpoint (`1400px`). |
