# DialogAnimation

Open/close animation of the `HxDialog`. .

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Fade | 0 | Fade on open and close (Bootstrap default). |
| None | 1 | No animation (`dialog-instant`). |
| SlideDown | 2 | Slides down from the top of the viewport (`dialog-slide-down`). |
| SlideUp | 3 | Slides up from the bottom of the viewport (`dialog-slide-up`). |
