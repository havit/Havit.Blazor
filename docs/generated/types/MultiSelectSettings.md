# MultiSelectSettings

Settings for the `HxMultiSelect` component.

## Properties

| Name | Type | Description |
|------|------|-------------|
| AllowFiltering | `bool` | Enables filtering capabilities. |
| AllowSelectAll | `bool` | Enables select all capabilities. |
| ClearFilterOnHide | `bool` | When enabled, the filter will be cleared when the dropdown is closed. |
| FilterClearIcon | `IconBase` | Icon displayed in the filter input for clearing the filter. |
| FilterSearchIcon | `IconBase` | Icon displayed in the filter input for searching the filter. |
| InputSize | `InputSize?` | Input size. |
| LabelType | `LabelType?` | The label type. |
| ValidationMessageMode | `ValidationMessageMode?` | Specifies how the validation message should be displayed. |
