# GridContentNavigationMode

Strategy for displaying and loading data in the grid.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Pagination | 1 | Use pagination. |
| LoadMore | 2 | Use "Load more" button. |
| PaginationAndLoadMore | 3 | Use pagination and "Load more" button. |
| InfiniteScroll | 4 | Use infinite scroll (virtualized). |
