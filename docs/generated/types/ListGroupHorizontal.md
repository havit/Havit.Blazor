# ListGroupHorizontal

Responsive horizontal setting (breakpoint) for `HxListGroup`. Default is `ListGroupHorizontal.Never`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Never | 0 | Never horizontal, always vertical. |
| SmallUp | 1 | Horizontal for viewports above the "small" breakpoint (`576px`). |
| MediumUp | 2 | Horizontal for viewports above the "medium" breakpoint (`768px`). |
| LargeUp | 3 | Horizontal for viewports above the "large" breakpoint (`992px`). |
| ExtraLargeUp | 4 | Horizontal for viewports above the "extra-large" breakpoint (`1200px`). |
| XxlUp | 5 | Horizontal for viewports above the "XXL" breakpoint (`1400px`). |
| Always | 6 | Always horizontal, never vertical. |
