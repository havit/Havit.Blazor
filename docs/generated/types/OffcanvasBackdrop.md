# OffcanvasBackdrop

Options for controlling the behavior of the `HxOffcanvas.Backdrop`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| True | 0 | A backdrop will be rendered. The offcanvas will be closed upon clicking on the backdrop. |
| False | 1 | No backdrop will be rendered. Users can interact with other parts of the app while the offcanvas is open. |
| Static | 2 | A static backdrop will be rendered. The offcanvas will not be closed upon clicking on the backdrop. |
