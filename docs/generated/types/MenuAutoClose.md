# MenuAutoClose

Auto-close behavior of the `HxMenu` (`HxMenuToggleButton` respectively). .

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| True | 0 | The menu will be closed by clicking outside or inside the menu. Default. |
| False | 1 | The menu will be closed by clicking the toggle button. (It will also not be closed by pressing the esc key). |
| Inside | 2 | The menu will be closed (only) by clicking inside the menu. |
| Outside | 3 | The menu will be closed (only) by clicking outside the menu. |
