# AvatarSize

Size for `HxAvatar` and `HxAvatarStack`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Regular | 0 | Regular (default) size. |
| ExtraSmall | 1 | Extra small avatar (`avatar-xs`). |
| Small | 2 | Small avatar (`avatar-sm`). |
| Large | 3 | Large avatar (`avatar-lg`). |
| ExtraLarge | 4 | Extra large avatar (`avatar-xl`). |
