# NavVariant

Variations for `HxNav`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Standard | 0 | Base `nav`. |
| Tabs | 1 | Tabs. https://getbootstrap.com/docs/5.3/components/navs-tabs/#tabs Remember to set the `active` tab (`HxNavLink.CssClass`). |
| Pills | 2 | Pills https://getbootstrap.com/docs/5.3/components/navs-tabs/#pills. |
| Underline | 3 | Underline https://getbootstrap.com/docs/5.3/components/navs-tabs/#underline. |
