# PlaceholderContainerSettings

Settings for the `HxPlaceholderContainer` and derived components.

## Properties

| Name | Type | Description |
|------|------|-------------|
| Animation | `PlaceholderAnimation?` | Animation of the placeholders in the container. |
| CssClass | `string` | Additional CSS class for the `HxPlaceholderContainer`. |
| ElementName | `string` | Gets or sets the name of the element to render. |
