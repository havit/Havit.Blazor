# CalendarDateCustomizationResult

Customization result for a specific date in the calendar (`HxCalendar`, `HxInputDate`, `HxInputDateRange`).

## Properties

| Name | Type | Description |
|------|------|-------------|
| CssClass | `string` | Specifies an additional CSS class for the date. |
| Enabled | `bool` | Indicates whether the date is enabled. The default value is `true`. |
