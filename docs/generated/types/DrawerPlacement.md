# DrawerPlacement

Represents the placement of an drawer.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| End | 0 | Drawer is placed at the end. |
| Start | 1 | Drawer is placed at the start. |
| Top | 2 | Drawer is placed at the top. |
| Bottom | 3 | Drawer is placed at the bottom. |
