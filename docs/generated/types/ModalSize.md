# ModalSize

Size for `HxModal`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Regular | 0 | Medium (default). Modal max-width: 500px. |
| Large | 1 | Large. Modal max-width: 800px. |
| ExtraLarge | 2 | Extra large. Modal max-width: 1140px. |
| Small | -1 | Small. Modal max-width: 300px. |
