# DialogSize

Size for `HxDialog`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Regular | 0 | Medium (default). Dialog max-width: 500px. |
| Large | 1 | Large. Dialog max-width: 800px. |
| ExtraLarge | 2 | Extra large. Dialog max-width: 1140px. |
| Small | -1 | Small. Dialog max-width: 300px. |
