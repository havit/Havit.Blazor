# AvatarStatus

Status indicator for `HxAvatar`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| None | 0 | No status indicator (default). |
| Online | 1 | Online status — green circle (`status-online`). |
| Offline | 2 | Offline status — gray rounded square (`status-offline`). |
| Busy | 3 | Busy status — red rounded square (`status-busy`). |
| Away | 4 | Away status — yellow circle (`status-away`). |
