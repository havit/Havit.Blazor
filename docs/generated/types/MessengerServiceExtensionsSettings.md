# MessengerServiceExtensionsSettings

Defaults for `HxMessengerServiceExtensions`.

## Properties

| Name | Type | Description |
|------|------|-------------|
| ErrorAutohideDelay | `int?` | Default autohide delay for errors (in milliseconds). Default is `null` (do not autohide). |
| ErrorColor | `ThemeColor` | Color scheme of error messages. |
| ErrorCssClass | `string` | CssClass of error messages. |
| InformationAutohideDelay | `int?` | Default autohide delay for information (in milliseconds). Default is `5000` ms. |
| InformationColor | `ThemeColor` | Color scheme of informational messages. |
| InformationCssClass | `string` | CssClass of informational messages. |
| WarningAutohideDelay | `int?` | Default autohide delay for warnings (in milliseconds). Default is `null` (do not autohide). |
| WarningColor | `ThemeColor` | Color scheme of warning messages. |
| WarningCssClass | `string` | CssClass of warning messages. |
