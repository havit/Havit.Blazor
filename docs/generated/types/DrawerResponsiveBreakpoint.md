# DrawerResponsiveBreakpoint

Content is displayed using the drawer when below this breakpoint, otherwise, contents are rendered in-place.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| None | 0 | Always render contents outside the viewport in an drawer. |
| Small | 1 | Render contents outside the viewport in an drawer when below the "small" breakpoint (`576px`, exclusive). |
| Medium | 3 | Render contents outside the viewport in an drawer when below the "medium" breakpoint (`768px`, exclusive). |
| Large | 4 | Render contents outside the viewport in an drawer when below the "large" breakpoint (`1024px`, exclusive). |
| ExtraLarge | 5 | Render contents outside the viewport in an drawer when below the "extra-large" breakpoint (`1280px`, exclusive). |
| Xxl | 6 | Render contents outside the viewport in an drawer when below the "2xl" breakpoint (`1536px`, exclusive). |
