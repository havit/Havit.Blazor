# StepperSettings

Settings for `HxStepper`.

## Properties

| Name | Type | Description |
|------|------|-------------|
| Color | `ThemeColor?` | Theme color of the stepper (renders the `theme-*` class). |
| CssClass | `string` | Additional CSS class(es) for the stepper. |
| Horizontal | `StepperHorizontal?` | Layout of the stepper items (vertical/horizontal, including the responsive breakpoint). |
| Overflow | `bool?` | When `true`, the stepper is wrapped in a scrollable `stepper-overflow` container. |
