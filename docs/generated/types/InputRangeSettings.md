# InputRangeSettings

Settings for the `HxInputRange` component.

## Properties

| Name | Type | Description |
|------|------|-------------|
| BindEvent | `BindEvent?` | Determines whether the `Value` is going to be updated instantly or `onchange` (usually `onmouseup`). |
| ValidationMessageMode | `ValidationMessageMode?` | Specifies how the validation message should be displayed. |
