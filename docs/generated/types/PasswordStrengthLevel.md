# PasswordStrengthLevel

Password strength level used by `HxPasswordStrength`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Weak | 0 | Weak password (one segment of the meter is filled). |
| Fair | 1 | Fair password (two segments of the meter are filled). |
| Good | 2 | Good password (three segments of the meter are filled). |
| Strong | 3 | Strong password (all four segments of the meter are filled). |
