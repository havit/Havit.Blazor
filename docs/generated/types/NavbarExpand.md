# NavbarExpand

Responsive expand setting (breakpoint) for `HxNavbar`.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Always | 0 | Always expanded, never collapses. |
| SmallUp | 1 | Expanded for viewports above the "small" breakpoint (`576px`). |
| MediumUp | 2 | Expanded for viewports above the "medium" breakpoint (`768px`). |
| LargeUp | 3 | Expanded for viewports above the "large" breakpoint (`1024px`). |
| ExtraLargeUp | 4 | Expanded for viewports above the "extra-large" breakpoint (`1280px`). |
| XxlUp | 5 | Expanded for viewports above the "2xl" breakpoint (`1536px`). |
| Never | 6 | Never expanded, always collapsed. |
