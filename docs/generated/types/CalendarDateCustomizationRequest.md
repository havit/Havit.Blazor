# CalendarDateCustomizationRequest

Customization request for a specific date in the calendar (`HxCalendar`, `HxInputDate`, `HxInputDateRange`).

## Properties

| Name | Type | Description |
|------|------|-------------|
| Date | `DateTime` | The date to be customized. |
| Target | `CalendarDateCustomizationTarget` | Who is requesting the customization and where it will be applied. Allows distinguishing between From and To inputs in `HxInputDateRange`. |
