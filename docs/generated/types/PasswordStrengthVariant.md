# PasswordStrengthVariant

Visualization variant of the `HxPasswordStrength` meter.

## Enum values

| Name | Value | Description |
|------|-------|-------------|
| Segments | 0 | Segmented meter—four segments that fill progressively as the password strength increases (renders the `strength` CSS class). |
| ProgressBar | 1 | Single progress bar that grows with the password strength (renders the `strength-bar` CSS class). |
