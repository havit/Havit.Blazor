# CheckboxSettings

Settings for `HxCheckbox`.

## Properties

| Name | Type | Description |
|------|------|-------------|
| ButtonSize | `ButtonSize?` | Size of the button for `CheckboxRenderMode.ToggleButton`. |
| Color | `ThemeColor?` | Color for `CheckboxRenderMode.ToggleButton`. |
| RenderMode | `CheckboxRenderMode?` | Checkbox render mode. |
| ValidationMessageMode | `ValidationMessageMode?` | Specifies how the validation message should be displayed. |
| Variant | `ButtonVariant?` | Indicates whether to use Bootstrap "outline" buttons. for `CheckboxRenderMode.ToggleButton`. |
