# HxIcon

Displays an icon. Currently supports Bootstrap icons through the `BootstrapIcon` class. Full documentation and demos can be found at https://havit.blazor.eu/components/HxIcon. You can easily add your own icon set.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| AdditionalAttributes | `Dictionary<string, object>` | Additional attributes to be splatted onto an underlying HTML element. |
| AriaLabel | `string` | Accessibility label for screen readers. Hides the icon from screen readers if not set. |
| CssClass | `string` | The CSS class to combine with the basic icon CSS class. |
| Icon | `IconBase` | The icon to display. |
