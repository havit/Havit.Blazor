# HxNavbar

Bootstrap 5 Navbar component - a responsive navigation header. Provides support for branding, navigation, and more, including support for the collapse plugin.

## Parameters

| Name | Type | Description |
|------|------|-------------|
| Color | `ThemeColor` | Color (background). The default value is `ThemeColor.None`. |
| ColorMode | `ColorMode` | Bootstrap theme (color mode) applied to the navbar. The default value is `ColorMode.Auto` (none). |
| ContainerCssClass | `string` | Container CSS class. The default value is `container-fluid`. |
| CssClass | `string` | Additional CSS class. |
| Expand | `NavbarExpand` | Responsive expand setting (breakpoint) for `HxNavbar`. The default value is `NavbarExpand.LargeUp`. |
| ChildContent | `RenderFragment` | Content of the navbar. |
| Id | `string` | Element ID. (An autogenerated GUID is used if not set explicitly.) |
